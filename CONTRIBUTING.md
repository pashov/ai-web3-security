# Contributing

Want to add a tool to the Web3 Security Tools Hub? We'd love your help!

## Adding a Free/Open Source Tool

1. Fork this repo
2. Add your tool link to the appropriate section in `README.md`
3. Open a pull request with a short description of the tool

## Suggesting a Paid Tool

Open an issue or PR with:
- Tool name and link
- Brief description of what it does

## Guidelines

- Tools must be related to web3 security
- Free tools should be open source with a public repo
- Keep entries concise — just the name and link
